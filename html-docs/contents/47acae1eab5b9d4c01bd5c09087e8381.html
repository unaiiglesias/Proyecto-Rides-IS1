Flujo de eventos: Depositar dinero

  1. El usuario indica la cantidad a depositar.
  2. El sistema almacena dicha cantidad y lo muestra en la cuenta.

Flujo alternativo

  • Si el usuario no introduce un numero positivo, el sistema no realiza la operación y lo indica.

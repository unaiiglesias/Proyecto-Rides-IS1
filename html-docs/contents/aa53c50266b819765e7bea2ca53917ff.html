Show Reviews

UMLUseCase
Untitled :: Requirements :: Show Reviews

Description

Flujo de eventos: Añadir reseña

  1. El viajero escoge un viaje completado.
  2. El sistema todas las reseñas de todos los viajeros que hayan completado ese viaje.

Flujo alternativo

  • Además, si así lo desea, el viajero podrá Valorar reseña.

Untitled :: Requirements :: Login

Description

Flujo de eventos: Hacer login

  1. El usuario indica email y contraseña
  2. El sistema busca la cuenta asociada a dicho email y contraseña y le asigna el rol correspondiente

Flujo de eventos alternativo:

  • Si no existe conductor o viajero que coincida con las credenciales proporcionadas, el sistema deniega el inicio de sesión

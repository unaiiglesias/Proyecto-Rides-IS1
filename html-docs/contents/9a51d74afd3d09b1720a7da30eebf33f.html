Accept Reservation

UMLUseCase
Untitled :: Requirements :: Accept Reservation

Description

Flujo de eventos: Aceptar petición de reserva

  1. El conductor selecciona uno de sus viajes.
  2. El sistema muestra todas las reservas de dicho viaje.
  3. Si el conductor acepta una solicitud, el sistema la marca como aceptada.

Flujo alternativo

  • Si el viajero solicita más asientos de los disponibles, la reserva no se acepta y el sistema lo indica.
  • Si no hay solicitudes pendientes, el sistema lo indica.

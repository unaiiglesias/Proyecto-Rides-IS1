Cancel Reservation Request

UMLUseCase
Untitled :: Requirements :: Cancel Reservation Request

Description

Flujo de eventos: Eliminar solicitud de viaje

  1. El sistema muestra todas las solicitudes hechas por el viajero.
  2. El viajero selecciona su solicitud y el sistema la elimina de la lista.

Flujo alternativo

2.1 Si la reserva está pagada, el sistema no la mostrará ni se podrá cancelar. 2.2 Si la reserva pertenece a un viaje con fecha pasada, el sistema no la mostrará.
